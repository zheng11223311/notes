### 1.Charles 抓手游模拟器数据包方式

Charles 官网:[Charles Web Debugging Proxy • HTTP Monitor / HTTP Proxy / HTTPS & SSL Proxy / Reverse Proxy (charlesproxy.com)](https://www.charlesproxy.com/)

**第二种方式**
