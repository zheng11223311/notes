### 1.vim 文本编辑器

三种模式

- 命令模式
- 插入模式
- 末行模式

```shell
vim index.html   #打开文件
#hjkl 	上下左右 光标移动
#再输入a (向后追加)或i(修改当前光标下的文字) 或o(开启新的一行,再新一行的头部) ,改变成插入模式,进行编辑
#再按Esc,退出至起初的打开模式
#按shift+冒号:    //进入末行模式
#输入wq  //保存退出  :w 保存   :q 退出  :wq! 强制保存退出(管理员使用,修改有权限的文本)  :q! 不保存退出


vim index.html   #打开文件
#按shift+冒号:    //进入末行模式
#输入set nu   //设置行号,显示行数
#输入 200     //跳转到200行
#按2下g   //移动到首行
#按下大G   //光标移动到最后一行 
#按2下y   //复制当前行,复制多行 nyy
#按下p   //粘贴
#按下x   //删除(也是剪切功能)字符,nx 删除多个字符,3x 删除3个
#按2下d   //删除行,ndd 删除多行,当前行向下删除
#按下u   //撤销   ctrl+r  反撤销
#输入set nonu   //取消
```

**建立配置文件**

```shell
vim ~/.vimrc      #~ 代表当前文件夹 .vimrc 为隐藏的配置文件
#按下a/i/o
#输入 set nu	#设置显示行数number,set unu 不显示行数
#输入wq 保存退出

#再打开其他文件时,都会显示文件的行数
```

**查找内容**

```shell
vim index.html
#按shift+冒号:    //进入末行模式
#输入/aa   //查找文件中 aa 字符串
#按下n     //切换到下一个匹配的值
#按下N     //切换到上一个匹配的值
```

**全文替换**

```shell
vim index.html
#按shift+冒号:    //进入末行模式
#输入 /aa   //查找文件中 aa 字符串
#输入 %s/aa/bb/g     //s 代表开始,g 代表结束  / 代表分隔符   将所有aa 替换成bb
```

**范围替换**

```shell
vim index.html
#按shift+冒号:    //进入末行模式
#输入/aa   //查找文件中 aa 字符串
#输入680,700s/aa/bb/g   //将680-700行之间的aa 替换成bb
```
