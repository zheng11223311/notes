### 1.跟时间相关的函数

**1.getDate()**	

作用: 获取当前时间的所有年月日等信息

返回值: 返回当前时间信息数组

参数:默认是当前时间戳

```php
<?php
echo getDate();
//或 getdate()
?>
```

**结果**

```
array (size=11)		
  'seconds' => int 16						秒
  'minutes' => int 37						分钟
  'hours' => int 7						    小时
  'mday' => int 15							月份中日 的表示

  'mon' => int 10							月份的数值表示
  'year' => int 2021						年份的数值表示

  'month' => string 'October' (length=7)	月份的完整文本表示

```

**2.mktime()  将指定时间转换成时间戳**

时分秒月日年

```php
<?php
//2020 年 11月 5 日 10时2分5秒  转换成时间戳
$timeUnix=mktime(10,2,5,11,5,2020);	
var_dump($timeUnix);	//1604570525
var_dump(Date('Y-m-d H:i:s',$timeUnix));	//2020-11-05 10:02:05

//时间溢出,溢出向后一天增加时间
$timeUnix=mktime(13,26,15,16,51,2020);	
var_dump($timeUnix);	//1621603575
var_dump(Date('Y-m-d H:i:s',$timeUnix));	//2021-05-21 13:26:15

//2020 年 11月 5 日 不需要时分秒,将值设置为0,不设置为默认当前时间
$timeUnix=mktime(0,0,0,11,5,2020);	
var_dump($timeUnix);	//1604534400
var_dump(Date('Y-m-d H:i:s',$timeUnix));	//2020-11-05 00:00:00
?>
```

**3.strtotime()  获取获取当前时间前或后多少天**

```php
<?php
//设置时间区
date_default_timezone_set("PRC");

//3天前日期
$nuix=strtotime('-3 day');
var_dump($nuix);
$date=date('Y-m-d',$nuix);	//也可以只写'Y',表示只获取年
var_dump($date);	//3天前日期

//3天后日期
$nuix=strtotime('3 day');
var_dump($nuix);
$date=date('Y-m-d',$nuix);	//也可以只写'Y-m',表示只获取年月
var_dump($date);	//3天后日期

//3年后日期
$nuix=strtotime('3 year');
var_dump($nuix);
$date=date('Y-m-d',$nuix);	//也可以只写'Y-m',表示只获取年月
var_dump($date);	//3年后日期

//3月后日期
$nuix=strtotime('3 month');
var_dump($nuix);
$date=date('Y-m-d',$nuix);	//也可以只写'Y-m',表示只获取年月
var_dump($date);	//3月后日期
?>
```
