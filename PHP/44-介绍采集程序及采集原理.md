**1.下载phpcms**

**下载UTF8 版本**

**注意:php 版本要高于5.4 才能用**

**2.安装**

在浏览器中打开到相对位置的 ./install_package/install/install.php 文件进行安装

出现提示:DNS  报错,解决方法

在install.php 文件中找到$PHP_DNS

```
 $PHP_DNS = preg_match("/^[0-9.]{7,15}$/", @gethostbyname('www.phpcms.cn')) ? 1 : 0;
```

修改为只要可以连上的域名就行,比如www.lmonkey.com

```
 $PHP_DNS = preg_match("/^[0-9.]{7,15}$/", @gethostbyname('www.lmonkey.com')) ? 1 : 0;
```

在浏览器中打开到相对位置的 ./install_package/install/install.php 文件进行安装
