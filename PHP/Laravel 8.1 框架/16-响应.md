# 响应

参考手册   [响应 | 基础功能 |《Laravel 8 中文文档 8.x》| Laravel China 社区 (learnku.com)](https://learnku.com/docs/laravel/8.x/responses/9370) 



**掌握内容**

- 字符串 & 数组
- 添加响应头
- 重定向
- 重定向并使用闪存的Session 数据
- 视图响应
- JSON 响应





**创建一个单函数控制器用于测试**

```
php artisan make:controller Test3Controller --invokable
```



`项目/routes/web.php`

```php
<?php
    
use App\Http\Controllers\Test3Controller;
use Illuminate\Support\Facades\Route;

Route::get('/test1',Test3Controller::class);
// Route::get('/test1/{id}',Test3Controller::class);
```
