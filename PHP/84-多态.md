### 1.多态

**概念:不同的人做相同的事得到不同的结果**

instanceof 运算符 判断一个对象是不是一个类的实例

```php
<?php
// 妹子
class Meizi{
    function meet($obj){
        //判断这个obj 类是不是通过hanzi 做出来的
        if($obj instanceof Hanzi){
            $obj->ber();
        }else{
            echo '我要的是哥哥';
        }
    }
    //或者在参数上直接限制类型,但是不是Hanzi 的类型还是会报错
    // function meet(Hanzi $obj){
    //         $obj->ber();
    // }
}

//汉子类
abstract class Hanzi{
    abstract function ber();
}

//掉石磊
class Ds extends Hanzi{
    public function AA(){
        echo '美女,我们AA 吧';
    }
    public function ber(){
        echo '啪';
    }
}
// 高富帅类
class Gfs extends Hanzi{
    public function flower(){
        echo '献上一朵小菊花';
    }
    public function ber(){
        echo '么么哒~';
    }
}

// 实例化
$meizi=new Meizi;
// 相亲对象
$ds=new Ds;
$gfs=new Gfs;

$meizi->meet($ds);
$meizi->meet($gfs);

//如果妹纸和妹纸相亲呢?
$ty=new Meizi();
// $meizi->meet($ty);  //报错
?>



```
