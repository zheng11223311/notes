### 1.伪静态规则

.htaccess 文件

```htaccess
#开启规则
RewriteEngine On
#规则作用范围在 / 及其子目录
RewriteBase /

# REQUEST_FILENAME 请求
# !-f  拦截请求为文件的请求,比如index.html
# !-d   拦截请求为目录的请求,比如/admin/
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d

#	不拦截的请求
RewriteCond %{REQUEST_URI} !^.*(\.css|\.js|\.png|\.gif|\.jpeg|\.jpg)



#   ^(.*)$   以(.*)开头且结尾的,(.*) 代表所有类型的文件或目录
#	将拦截的请求转到idnex.php 文件
#	index.php?$1  $1代表将(.*) 的内容, 即将$1 作为index.php 的参数
#	 [QSA,L] 其他规则
RewriteRule ^(.*)$ index.php?$1 [QSA,L]
```
