### 1.模块化

- 每个文件被视为一个独立的模块
- 绝对路径 /
- 相对路径 ./
- 文件夹
- module.exports
- node_modules 安装包
- API 文档
  - https://nodejs.cn/api/

**示例:**

```js
//引入相对路径
require('./2.js')

//引入绝对路径,在linux 中使用pwd 查看当前路径
reqire('/usr/local/node/3.js') 

//引入目录,自动找文件夹下面的index.js 文件
require('./libs')

//引入文件中包含函数,或者对象
require('./func.js')	//不能直接使用文件中的函数/对象,文件中需要导出函数/对象才能被使用

//引入npm 工具包
var formidable=require('formidable')
```

**func.js**

```js
function(m,n){
```
