### 1.收尾工作

**添加环境变量**

```
echo $PATH   //使用命令查看环境变量
显示
/usr/local/sbin:/usr/local/bin:/sbin:/bin:/usr/sbin:/usr/bin:/root/bin

export RATH=/usr/local/php/bin:$PATH   //添加环境变量,临时性,重启时失效
echo $PATH
显示
/usr/local/php/bin:/usr/local/php/bin:/usr/lib/qt-3.3/bin:/usr/local/sbin:/usr/local/bin:/sbin:/bin:/usr/sbin:/usr/bin:/root/bin

vim /etc/profile     //添加环境变量,永久性
//在最后一行加上
export PATH="/usr/local/php/bin:$PATH"   //php 环境变量
export PATH="/usr/local/mysql/bin:$PATH"  //mysql 环境变量
export PATH="/usr/local/apache/bin:$PATH"  //apache 环境变量

source /etc/profile   //执行文件
mysql -u root  -p    //执行文件后这个命令才有效
php -v               //执行文件后这个命令才有效
apachectl -v		//执行文件后这个命令才有效
```

**安装openssl**

OpenSSL 是一个强大的安全套接字层密码库,囊括主要的密码算法,常用的密钥和证书封装管理功能及SSL 协议,并提供丰富的应用程序供测试或则其他目的使用

```
yum -y install openssl-devel    //必须安装

cd /lamp/php-7.0.7/ext/openssl   //openssl 的安装是要进入扩展ext 文件中
mv config0.m4  config.m4        //否则报错:找不到config.m4
/usr/local/php/bin/phpize       //执行此语句,生成configure 命令来进行检测,其他第三方模块都需要使用这种命令来生成configure 配置文件
./configure --with-openssl --with-php-config=/usr/local/php/bin/php-config
make
make install

修改 /usr/local/php/etc/php.ini
extension_dir="/usr/local/php/lib/php/extensions/no-debug-zts-20151012"
//打开注释,并修改,不是On_windows 下的那个(liunx 的)

extension="openssl.so";     //so 动态模块文件
//添加

重启apache,在phpinfo() 中可以找到这个模块
apachectl stop
apachectl start
```

**安装phpMyAdmin**

```

cp config.sample.inc.php   config.inc.php

$cfg['Servers'][$i]['auth_type']='cookie';  //不够安全
```
