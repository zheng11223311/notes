### 1.单例设计模式

最终的结果只能实例化一个对象

```php
<?php
class Mysql{
    static $link=null;
    private function __construct(){
        echo '与数据库建立了一次连接';
    }
    static public function connect(){
        if(is_null(self::$link)){
            self::$link=new self;
        }
        return self::$link;
    }

}

// 实例化
// $mysql=new Mysql;

//不论调用多少次connect 都执行一次输出,只能实例化一次
Mysql::connect();	//与数据库建立了一次连接
Mysql::connect();
Mysql::connect();
Mysql::connect();
?>



```
