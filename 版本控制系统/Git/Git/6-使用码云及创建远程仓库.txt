1.注册登录码云，完善个人信息
2.设置本机绑定SSH 公钥匙，实现免密码登入！（免密码登入，这一步挺重要的，码云是远程仓库，我们平时是工作在本地仓库！）

#进入 C:\users\administractor\.ssh 目录（只要安装了git 都有这个目录）
没有这个目录自己创建 mkdir .ssh
#生成公钥
ssh-keygen
或使用算法加密，比如rsa 算法
ssh-keygen -t rsa

一路回车，生成2个文件（公共id_rsa.pub和 私钥id_rsa）
复制公共id_rsa.pub 里面的内容（公钥），粘贴到码云公钥上

3.将公钥信息public key 添加到码云账户中即可！
4.使用码云创建一个自己的仓库！
