**变量的解构赋值**

ES6 允许按照一定模式从数组和对象中提取,对变量进行赋值,这被称为解构赋值

**1.数组的解构赋值**

```js
const a=['a','b','c','d']
let [a1,b1,c1,d1]=a
console.log(a1,b1,c1,d1);
```

**2.对象的解构赋值**

```js
const b={
           name:'tom',
           age:18,
           fn:function(){
               console.log('这是一个函数');
           }
       }
       let {name,age,fn}=b
       console.log(name,age,fn);
```
