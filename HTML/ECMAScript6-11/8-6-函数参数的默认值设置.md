**ES6 允许函数参数赋值初始值**

**1.形参初始值 具有默认的参数,一般位置要靠后(潜规则)**

```js
function add(a,b=20){
        return a+b
    }
    add(10)
```

**默认值也可以是一个变量**

```js
const init=20
function add(a,b=init){
        return a+b
    }
    add(10)
```

**2.与解构赋值结合使用**

```
function add({name,age,hobby='干饭'}){
        console.log(name,age,hobby)
    }
    add({
    name:'Tom',
    age:18
    })
```
