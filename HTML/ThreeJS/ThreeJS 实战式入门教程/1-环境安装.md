**执行安装**

```powershell
cd threejs			#进入项目
npm install			#安装依赖包
npm run dev			#启动,或者使用 nodemon run dev 热跟新,这里的vite 就是热更新的,可以不用使用nodemon
```

**浏览器访问**

localhost:3000



**安装threejs**

```
npm i three
```

在使用的时候如果出`import`导入`three` 失败报红,请使用

```
npm i --save-dev @types/three
```
