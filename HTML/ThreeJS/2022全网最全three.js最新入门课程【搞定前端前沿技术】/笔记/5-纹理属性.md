## 1-纹理属性

```js
//导入纹理
const textureLoader=new THREE.TextureLoader()  //实例化
const texture=textureLoader.load('1.jpg') //加载纹理图片

texture.offset.x=0.5	//纹理偏移0-1,像背景图偏移一样
texture.offset.y=0.5
texture.offset.set(0.5,0.5)


//设置旋转中心
texture.center.set(0.5,0.5)  //x,y
//纹理的旋转 弧度单位
texture.rotation.x=Math.PI  //默认以一个顶点为中心旋转
//纹理的重复
texture.repeat.set(2,3) //水平重复2次,竖直重复3次,默认纹理重复的模式是将纹理边缘向外推
//设置纹理重复的模式
texture.wrapS=THREE.RepeatWrapping  //wrapS uv 中的水平方向 THREE.RepeatWrapping 纹理常量 无穷重复
texture.wrapT=THREE.MirroredRepeatWrapping   //wrapT uv 中的垂直方向 MirroredRepeatWrapping 镜像重复

```



## 2-设置纹理显示的算法与mipmap

`magFilter`

当纹理覆盖大于一个像素是,贴图如何采样

`mipFilter`

当纹理覆盖小于一个像素是,贴图如何采样

```js
texture.minFilter=THREE.NearestFilter  //最接近旁边的颜色 ,THREE.LinearFilter 将最近的颜色值取平均值作为它的值
texture.magFilter=THREE.NearestFilter
```



## 3-透明材质与透明纹理

```js
const basicMaterial=new THREE.MeshBasicMaterial({
    color:"#ffff00", //材质颜色
    //map:texture,	//指定使用的纹理,
    alphaMap:texture,	//使用透明纹理,将纹理透明化,需要先开启允许透明
    transparent:true,	//开启透明
    //opacity:0.5,	//指定透明度
    side:THREE.DoubleSide,	//指定渲染平面的两面(前面和后面),为性能考虑,默认值为渲染前面,因为后面不需要被看到
})

basicMaterial.side=HREE.DoubleSide  //也可以设置渲染的面
```



## 4-环境遮挡贴图(遮挡住环境光)与强度

使暗下去的线条更加暗,使贴图纹理更加清晰,更加真实,更加有凹凸感

```js
const plane=new THREE.PlaneBufferGeometry(1,1)  //平面的骨架
const basicMaterial=new THREE.MeshBasicMaterial({
    aoMap:texture,	//环境遮挡贴图设置
    aoMapIntensity:0.5, //设置遮挡强度
})
const mesh=new THREE.Mesh(plane,basicMaterial)

//需要给此物体的骨架重新设置新的uv ,相当于重新复制原来的uv
plane.setAttribute('uv2',new THREE.BufferAttribute(plane.attributes.uv.array,2))  //plane.attributes.uv.array 获取原来的uv, 2表示以2个轴为uv 坐标,因为uv 是一个二维平面坐标轴
```
