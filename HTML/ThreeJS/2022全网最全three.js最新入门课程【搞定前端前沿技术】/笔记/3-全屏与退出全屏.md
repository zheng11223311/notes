## 1-全屏与退出全屏

```js
window.addEventListener("dblclick",()=>{
    //双击控制屏幕进入全屏,退出全屏
    renderer.domElement.requestFullScreen() //进入全屏
    console.log(document.fullscreenElement) //查看全屏的元素 为canvas,没有则为null
    if(document.fullscreenElement){
        document.exitFullScreen() //退出全屏 使用document 对象
    }
})
```



## 2-dat 图形控制

`npm 地址` [dat.gui - npm (npmjs.com)](https://www.npmjs.com/package/dat.gui) 

安装

```
npm install dat.gui
```

使用

```js
import * as dat  from 'dat.gui'
const gui=new dat.GUI()
//控制属性 cube 的x 轴位置,滑动类型
gui.add(cube.position,'x').min(0).max(5).step(0.1).name('属性别名名称').onChange(value=>{
console.log('值被修改')
})
.onFinishChange(value=>{
console.log('鼠标抬起时,完全停下来触发')
})

const params={
    color:"#ccc",
    fn:()=>{
        console.log("被触发")
    }
}
//控制颜色, 颜色选择器
gui.addColor(params,'color').onChange(value=>{
    cube.material.color.set(value)
})
//控制物体显示,选项框
gui.add(cube,'visible').name('是否显示物体')
//点击类型
gui.add(params,'fn')

//可折叠文件夹
const folder=gui.addFolder()  //相当于一个gui
folder.add(params,'fn')
// 尝试文件夹中添加文件夹

```
