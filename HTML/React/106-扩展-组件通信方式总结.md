**9.组件通信方式总结**

**组件间的关系**

父子组件

兄弟组件（非嵌套组件）

祖孙组件（跨级组件）

**几种通行方式：**

```
1.props
	1.children pros
	2.render props
2.消息订阅-发布：
	pub-sub，event 等等
3.集中式管理：
	redux，dva 等等
4.contxt：
	生产者-消费模式（provider）
```

**比较好的搭配方式:**

```
父子组件：props
兄弟组件：消息订阅-发布，集中式管理
祖孙组件（跨级组件）：消息订阅-发布，集中式管理，context（开发中用的少，封装插件用的多）
```
