### 1.模块与依赖

webpack 模块

**支持**

1. ECMAScript 模块
2. CommonJS 模块
3. AMD 模块
4. Assets 模块

```js
import a from './a'

const a=require('./a')

define([],function(){return ()=>{} })

require(['./a.js','./b.js'],function(a,b){})

@import './a.less'
```

```css
.block{
    background:url('./a.jpg')!important;
}
```
