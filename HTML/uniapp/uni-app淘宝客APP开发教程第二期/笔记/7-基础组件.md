# 基础API 调用

地址: [系统信息的概念 | uni-app官网 (dcloud.io)](https://uniapp.dcloud.io/api/system/info.html) 

`pages/index/index.vue`

```vue
<script>
	export default {
		// 数据初始化
		data() {
			return {
				title: 'Hello'
			}
		},
		onLoad() {
			// 获取系统信息
			uni.getSystemInfo({
				success:(e)=>{
					console.log('这里是成功的回调,',e);
				},
				fail:(e)=> {
					console.log('这里是失败的回调',e);
				},
				complete:(e)=>{
					console.log('不论成功或失败都会执行的回调',e);
				}
			})
		},
		methods: {

		}
	}
</script>

```
