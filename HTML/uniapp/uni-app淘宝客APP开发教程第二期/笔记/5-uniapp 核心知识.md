# 条件编译

| 条件编译写法                                               | 说明                                                         |
| ---------------------------------------------------------- | ------------------------------------------------------------ |
| #ifdef APP-PLUS<br />需要条件编译的代码<br />#endif        | 仅出现在App平台下的代码                                      |
| #ifnef H5<br />需要条件编译的代码<br />#endif              | 除了H5 平台,其他平台均存在的代码                             |
| #ifdef H5\|\|MP-WEIXIN<br />需要条件编译的代码<br />#endif | 在H5 平台或微信小程序平台存在的代码(这里只有\|\|,不可能出现&& ,因为不可能同时出现) |
