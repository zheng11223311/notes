### 组件通讯

**父组件给子组件传值**

通过props 来接收外界传递到组件内部的值

**使用login 组件并传值**

```vue
<template>
	<view class="content">
		<!-- 使用login 组件,并传递da 的值给login 的msg 属性-->
		<login :msg='da'></login>
	</view>
</template>

<script>
	import login from '@/components/login/login.vue'
	export default{
		data(){ 
			return{
				da:'哈哈'
			}
		},
		components:{
			login
		},
		onLoad(options) {
			// 接收到get 传递的参数,其他导航跳转到此页面携带的参数
			console.log(options)
		},
		onUnload() {
			console.log('页面卸载了')
		}
		
	}
</script>

<style>
</style>
```

**login.vue**

```
<template>
	<view class="">
		这是login 组件
		使用msg 接收到的值,与data 一样使用
		{{msg}}	
	</view>
</template>

<script>
</script>
export default{
	props:['msg']	//接收传递过来的值
}
<style>
</style>
```

**子组件向父组件传递数据**

通过$emit 触发事件进行传递参数

**使用login 组件并接收值**

```vue
<template>
	<view class="content">
		<!-- 使用login 组件,在login 组件内接收传递过来的fun,将funct 作为函数名使用-->
		<login @fun="funct"></login>
	</view>
</template>

<script>
	import login from '@/components/login/login.vue'
	export default{
		data(){ 
			return{
				
			}
		},
		components:{
			login
		},
		onLoad(options) {
			// 接收到get 传递的参数,其他导航跳转到此页面携带的参数
			console.log(options)
		},
		onUnload() {
			console.log('页面卸载了')
		},
        methods:{
            funct(data){
                //data 为子组件传递过来的数据
                console.log(data)
            }
        }
		
	}
</script>

<style>
</style>
```

**login.vue**

```
<template>
	<view class="">
		这是login 组件
		<button @click='func'>向父组件传递值 </button>
	</view>
</template>

<script>
</script>
export default{
data(){ 
			return{
				da:'哈哈'
			}
		},
	methods:{
	func(){
	// $emit('父组件接收的值',传递的数据)
		this.$emit('fun',this.da)
	}
	}
}
<style>
</style>
```
