### 1.查看和编辑文件

地址:[应用 - App (autojs.org)](https://pro.autojs.org/docs/#/zh-cn/app?id=appviewfilepath)

## [app.viewFile(path)](https://pro.autojs.org/docs/#/zh-cn/app?id=appviewfilepath)

- `path` {string} 文件路径

用其他应用查看文件。文件不存在的情况由查看文件的应用处理。

如果找不出可以查看该文件的应用，则抛出`ActivityNotException`。

```
//查看文本文件
app.viewFile("/sdcard/1.txt");
```

## [app.editFile(path)](https://pro.autojs.org/docs/#/zh-cn/app?id=appeditfilepath)

- `path` {string} 文件路径

用其他应用编辑文件。文件不存在的情况由编辑文件的应用处理。

如果找不出可以编辑该文件的应用，则抛出`ActivityNotException`。

```
//编辑文本文件
app.editFile("/sdcard/1.txt/);
```

**示例:**

```js
// 查看文件
// 弹出选择查看文件的编辑器,有的会根据类型自动打开
// app.viewFile("/sdcard/1/UI上帝模式.js")
// 直接查看图片,使用默认(如果已经默认)图片查看器
// app.viewFile("/sdcard/1/index.jpg")


//没有响应
app.editFile("/sdcard/1/index.js");
// 响应编辑图片
// app.editFile("/sdcard/1/index.jpg");
```
