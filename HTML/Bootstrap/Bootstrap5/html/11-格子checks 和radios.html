当通过 JavaScript 手动设置时，复选框可以使用 :indeterminate 伪类（没有可用的 HTML 属性来指定它）
禁用（Disabled） 添加 disabled 属性，关联的 <label> 会自动设置样式以匹配较浅的颜色，以帮助指示input的状态。
Radios
Disabled 添加 disabled 属性，关联的 < label> 会自动设置样式以匹配较浅的颜色，以帮助指示input的状态。
开关（Switches） 开关具有自定义复选框的标记，但使用 .form-switch 类来呈现切换开关。 开关也支持 disabled(禁止) 属性。 需要继承基类form-check
Inline（在线） 通过在任何.form-check中添加.form-check-inline，将复选框或radios分组水平对齐。默认是垂直平分
无标签 对于没有标签文本的复选框和radios，省略包装的.form-check。记住仍然要为辅助技术提供某种形式的标签（例如，使用aria-label）
切换按钮（Toggle buttons） 复选框切换按钮（Checkbox toggle buttons） 通过使用.btn样式而不是.form-check-label在< label>元素上创建类似按钮的复选框和单选按钮。如果需要的话，这些切换按钮可以进一步被分组到一个按钮组中。
