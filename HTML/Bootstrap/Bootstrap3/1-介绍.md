# 介绍

官网: [Bootstrap中文网 (bootcss.com)](https://www.bootcss.com/) 



`bootstrap`为了增强跨浏览器表现的一致性,使用`Normalize.css` 初始化重置样式



## 布局

- 流体布局
- 固定布局
  - 宽的阈值                                     width 
  - 大于等于1200 px(lg 大屏PC)                  ->1170px(1140+槽宽) 槽宽:左右padding 之和,默认共30px
  - 大于等于992 px 小于等于1200(md 中屏PC)       ->970px(940+槽宽)

  - 等于768 px 小于等于992(sm 平板)          ->750px(720+槽宽)
  - 小于768(xs 移动手机)                         ->auto : 使用padding时,不会增加宽度,与100%的区别.100% 使用padding 会增加宽度
- 栅格系统



## 栅格系统源码分析

 栅格系统`less` 源码: ` ../less/grid.less `

所有`less` 定义的变量位置:`../less/variable.less`

列的定义 文件在 `../less/grid-framework.less`

- ​    **固定容器** 和 **流体容器** 的公共样式

  ```css
  margin-right:auto;
  
  margin-left:auto;
  
  padding-left:15px;
  
  padding-right:15px;
  ```

- **固定容器 **特定样式

  ```css
  //顺序不可变
  @media(min-width:@screen-sm-min){		//@screen-sm-min sm 的尺寸大小992
  	width:@container-sm;				//@container-sm  sm 对应的宽度
  }
  @media(min-width:@screen-md-min){
  	width:@container-md;
  }
  @media(min-width:@screen-lg-min){
  	width:@container-lg;
  }
  不满足上方条件的默认就是超小屏的xs
  ```

- 行

  ```css
  make-row(@gutter:@grid-gutter-width){
      margin-left:ceil((@gutter/-2));		// margin-left 为-15px
      margin-right:floor((@gutter/-2));	// margin-right  为-15px
      &:entend(.clearfix all);	//继承清除浮动样式 ../less/clearfix.less
  }
  ```

- 列

  ```less
  //列的定义 文件在 ../less/grid-framework.less
  .make-grid-cloumns();	
  .make-grid(xs);				//bootstrap 移动设备优先
  @media(min-width:@screen-sm-min){		//@screen-sm-min sm 的尺寸大小992
  	.make-grid(sm);				//@container-sm  sm 对应的宽度
  }
  @media(min-width:@screen-md-min){
  	.make-grid(md);
  }
  @media(min-width:@screen-lg-min){
  	.make-grid(lg);
  }
  ```

  ```less
  .make-grid-cloumns();	//解读,生成内容如下
  .col-xs-1,.col-sm-1,.col-md-1,.col-lg-1,
  .col-xs-2,.col-sm-2,.col-md-2,.col-lg-2,
  ...
  .col-xs-12,.col-sm-12,.col-md-12,.col-lg-12{
      position:relative;
      min-height:1px;
      pading-left:15px;
      padding-right:15px;
  }
  ```

  ```less
  .make-grid(xs);		 	//解读,里面又调用混合
  .float-grid-columns(@class);	//@class 为xs, 如果.make-grid() 中为其他参数,就是其他的列(lg,md,sm),为所有的.col-xs- 创建float:left 样式
  .loop-grid-columns(@grid-columns,@class,width);		//设置宽度
	.loop-grid-columns(@grid-columns,@class,pull);		//列排序
  .loop-grid-columns(@grid-columns,@class,push);		//列排序
  .loop-grid-columns(@grid-columns,@class,offset);	//列偏移
  ```
  
  ```css
  .float-grid-columns(@class);	// 此时@class 为xs,解读,生成内容如下
  .col-xs-1,
  .col-xs-2,
  ...
  .col-xs-12{
     float:left
  }
  ```
  
  ```css
  .loop-grid-columns(@grid-columns,@class,width); //解读,生成内容如下
  .col-@{class}-@{index}{		//class 为xs ,index 为12
      width:percentage((@index/@grid-columns));	//percentage 转化为百分比 @grid-columns 为栅格系统的12列 
  }
  即会转化为如下
  .col-xs-12{
      width:12/12;
  }
  .col-xs-11{
      width:11/12;
  }
  .col-xs-10{
      width:10/12;
  }
  ...
  .col-xs-1{
      width:1/12
  }
  
  ```
  
  ```css
  .loop-grid-columns(@grid-columns,@class,pull);  //解读,生成内容如下
  .col-xs-push-12{
      left:12/12;
  }
  .col-xs-push-11{
      left:11/12;
  }
  .col-xs-push-10{
      left:10/12;
  }
  ...
  .col-xs-push-1{
      left:1/12
  }
  
  .col-xs-push-0{
      left:auto
  }
  
  ```
  
  ```css
  .loop-grid-columns(@grid-columns,@class,pull);  //解读,生成内容如下
  .col-xs-push-12{
      right:12/12;
  }
  .col-xs-push-11{
      right:11/12;
  }
  .col-xs-push-10{
      right:10/12;
  }
  ...
  .col-xs-push-1{
      left:1/12
  }
  .col-xs-push-0{
      right:auto
  }
  ```
  
  ```css
  .loop-grid-columns(@grid-columns,@class,offset);  //解读,生成内容如下
  .col-xs-offset-12{
      margin-left:12/12;
  }
  .col-xs-offset-11{
      margin-left:11/12;
  }
  .col-xs-offset-10{
      margin-left:10/12;
  }
  ...
  .col-xs-offset-1{
      margin-left:1/12;
  }
  .col-xs-offset-0{
      margin-left:0;
  }
  ```
