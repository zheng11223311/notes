协议:FTP HTTP STMP TCP/IP

        请求方法:
        GET 获取一个URL 指定的资源,既资源实体
        POST 向服务器提交资源
        PUT 向服务器提交资源
        DELETE 请求服务器删除Request-URL 标志的资源
        HEAD 获取一个指定资源的头信息
        OPTIONS 获取服务器支持的方法

        状态码:
        状态码  定义        说明
        1xx   请求被接受    一般只在实验环境下使用
        2xx    成功         操作成功的收到,理解和接收
        3xx     重定向      为了完成请求,进行进一步措施(服务器处理不了,提交到别处地址)
        4xx     客户端错误  请求的语法有错误或则不能完全被满足
        5xx     服务器错误  服务器无法完成明显有效的请求
