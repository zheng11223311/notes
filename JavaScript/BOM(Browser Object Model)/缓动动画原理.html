缓动效果原理
		缓动动画就是让元素运动速度有所变化,最常见的是让速度慢慢停下来

		思路
		1.让盒子每次移动的距离慢慢变小,速度就会慢慢落下来.
		2.核心算法:(目标值-现在的位置)/10 做为每次移动的距离步长
		3.停止的条件是让当前盒子位置等于目前位置就停止定时器

		有小数的步长会出现到达不到目标位置的情况
		走过的路不能倒回去,应该将小数步长往上取整.
		负数就向下取整
		Math.ceil(小数);
		Math.floor(小数); 往下取整

		fn(形参1,形参2,形参3){}
		fn(1,2,function(){
		xx.style.color='';
	});
	function(){} 为回调函数,当前面的形参执行完在执行回调函数
