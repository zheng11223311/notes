1.代码分类(位置)
                全局代码
                函数(局部)代码
            2.全局执行上下文
                在代码全局代码前将winodw 确定为全局执行上下文
                对全局数据进行预处理
                    var 定义的全局变量==> undefied,添加为wondow 的属性
                    function 声明的全局函数==> 赋值(fun) 添加为wondow 的方法
                    this==> 赋值(winodw)
                    开始执行全局代码
            3.函数执行上下文
                在调用函数,准备执行函数之前,创建对应的函数执行上下文对象
                对局部数据进行预处理
                    形参变量==> 赋值(实参)==>添加为执行上下文的属性
                    argument==> 赋值(实参列表),添加为执行上下文的属性
