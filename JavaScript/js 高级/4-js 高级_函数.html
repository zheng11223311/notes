1.什么是函数?
            实现特定功能的n 条语句块的封装体
            只有函数是可以执行的,其他类型的数据不能执行

        2.为什么要用函数?
            提高代码复用
            便于阅读交流

        3.如何定义函数?
         函数声明
            function fn(){

            }
        表达式
            var fn2=function(){
                变量提升
            }

        4如何调用(执行)函数?
            text(); 直接调用
            obj.text();  通过对象调用
            new text();   new 调用
            text.call / apply(obj): 临时让text 成为obj 的方法进行调用
