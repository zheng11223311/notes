        相关问题
        1.undefined 与null 的区别
        undefined 代表定义为赋值 --- var a;
        null 定义并赋值了,只是值为null --- var a; a=null;

        2.什么时候给变量赋值为null ?
        初始赋值,表明将要赋值为对象
            变量不在使用时,将对象成为垃圾对象被释放回收
            var a; a=1; a=null;

        3.严格区别变量类型与数据类型?
        数据类型
            基本类型
            对象类型

        变量类型(变量内存值的类型)
            基本类型: 保存就是基本类型的值
            引用类型: 保存的是地址值
